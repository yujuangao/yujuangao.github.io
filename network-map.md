# Mobile Network Coverage Analysis

This dashboard visualizes the expansion of mobile network generations (2G, 3G, and 4G) and detailed 3G cluster coverage from my research data.

### Key Research Features
* **Longitudinal Data:** View network expansion from 2012 to 2021.
* **Multi-Buffer Analysis:** Explore 3G coverage proportions across various buffer distances (0.5km to 100km).
* **Cross-Country Comparison:** National average coverage aggregated by technology generation.
